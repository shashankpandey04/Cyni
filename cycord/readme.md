# CyCord

**CyCord** is a custom helper module for **CYNI** that enables early adoption of unreleased features in `discord.py`.  
It provides lightweight wrappers around Discord’s HTTP API so CYNI can stay ahead of official library support.

---

## 📂 Structure
CyCord
├─ docs/
│ └─ # Documentation for Guild Self Profile API
│
└─ methods/
│ └─ # Implementation of Guild Self Profile API wrapper

---

## ⚠️ Disclaimer
CyCord is not an official extension of discord.py.
It is maintained for CYNI’s internal use and may change as upstream adds official support.
External adoption should be done with caution.
